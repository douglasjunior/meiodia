## Installing
Install globally:

    npm install -g meiodia

## Running
Run command:

    meiodia

Or import in NodeJS script:

    import meiodia from 'meiodia';

    meiodia();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx meiodia
